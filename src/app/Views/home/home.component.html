Enhance your Traktor library!

Why was this app created on the first place?

OLMA is a web application built with Angular, Google's front-end framework.

It's purpose is to enhance the Traktor library management experience about which many have complained on its official forums.

Despite it's improvments over the last year, the current state for managing your library is not optimial, yet. This app pretends to solve some flaws that many user's have been criticising about the software on its forums.

What does OLMA currently offer?

  • Provides a way to manage your Traktor library from anywhere, thanks to its connections to Cloud storage services.
  • Allows Traktor users to create smartlists which have more than 8 rules.
  • Connects to Spotify to create smartlists with enhanced metadata obtained by Spotify through Machine Learning algorithms.
